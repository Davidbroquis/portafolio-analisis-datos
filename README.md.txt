# Análisis de Tráfico Portuario Marítimo — Colombia 2018-2025

## Descripción
Proyecto de análisis de datos usando información oficial del tráfico 
portuario marítimo de Colombia, obtenida de datos.gov.co 
(Superintendencia de Transporte).

Este es mi primer proyecto de portafolio, desarrollado de forma 
autodidacta con el objetivo de crecer hacia un perfil de analista 
de datos.

## Herramientas utilizadas
- MySQL Workbench — limpieza de datos y consultas SQL
- Power BI Desktop — visualización y construcción de dashboard

## Preguntas de negocio respondidas
1. ¿Qué zona portuaria mueve más carga?
2. ¿Qué tipo de carga predomina en los puertos colombianos?
3. ¿Cómo ha evolucionado el tráfico portuario año a año (2018-2025)?
4. ¿En qué mes se concentra más actividad portuaria?
5. ¿Cuál sociedad portuaria tiene más comercio exterior?

## Hallazgos principales
- Barranquilla lidera en volumen total de comercio exterior
- El Granel Sólido es el tipo de carga más movilizado
- Se evidencia una caída en 2020 por el impacto del COVID-19,
  con recuperación sostenida desde 2021
- Noviembre es el mes de mayor actividad portuaria
- Las importaciones superan consistentemente a las exportaciones

## Fuente de datos
[Tráfico Portuario Marítimo - datos.gov.co](https://www.datos.gov.co/Transporte/Trafico-Portuario-Mar-timo-En-Colombia/5r3g-zv5z)
