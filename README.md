## 📝 Descripción del Proyecto
Este proyecto presenta un análisis profundo sobre el movimiento logístico de Colombia, utilizando datos oficiales de la **Superintendencia de Transporte**. Se aplicó un flujo de trabajo **ETL (Extract, Transform, Load)** completo para asegurar la integridad de **9,111 registros**, corrigiendo errores de codificación (UTF-8) y normalizando formatos de fecha para un análisis preciso.

---

## 🛠️ Stack Tecnológico
* **Excel:** Limpieza de datos, normalización y tratamiento de caracteres especiales.
* **MySQL Workbench:** Modelado de base de datos relacional y consultas de agregación.
* **Power BI Desktop:** Modelado de datos (DAX) y creación de tablero interactivo.

## 🔍 Preguntas de Negocio Respondidas (SQL Insights)

| Pregunta | Hallazgo Clave |
| :--- | :--- |
| **¿Zona portuaria líder?** | **Ciénaga** encabeza la lista con **284M** de toneladas (Tráfico de carbón). |
| **¿Carga predominante?** | El **Carbón al Granel** es el motor con **504.7M** de toneladas totales. |
| **¿Evolución temporal?** | Pico en 2018, caída en 2020 (Pandemia) y estabilidad en **140M** ton/año desde 2021. |
| **¿Mes de mayor flujo?** | **Marzo** es el mes con mayor actividad histórica (**107.9M** de toneladas). |
| **¿Sociedad líder?** | **Oleoducto Central S.A.** (Exportación) y **S.P. Regional de Buenaventura** (Carga mixta). |

---

## 📈 Hallazgos Principales (Data Storytelling)

* **⚖️ Balance Comercial:** Las exportaciones superan a las importaciones en volumen total de toneladas, debido al peso estratégico de materias primas como el carbón y el petróleo.
* **📍 Liderazgo Regional:** **Buenaventura** se consolida como el puerto público más importante para la economía nacional por su diversidad de carga y volumen de comercio exterior (**69.7M** de toneladas).
* **🎯 Especialización:** Los puertos colombianos muestran una alta especialización: Ciénaga en minerales, mientras que Cartagena y Buenaventura dominan en contenedores y carga general.
